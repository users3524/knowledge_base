# 软件架构与技能

这里沉淀嵌入式软件的底层驱动、实时系统、控制算法、架构模式和调试经验。适合放从具体项目中抽象出来、未来能复用的软件知识。

## 现有文档

- [嵌入式软件课程大纲](<嵌入式软件课程大纲.md>)

## 适合收纳

- 底层驱动、寄存器、HAL/LL、BSP 和 Bootloader。
- RTOS、任务调度、同步通信和系统调试。
- 控制算法、状态机、滤波、参数整定和实时性优化。

## 沉淀建议

- 驱动、RTOS、算法类内容优先放入对应子目录。
- 跨模块的软件架构、接口设计、状态机、任务划分和错误处理策略可放在本目录。
- Bug 复盘建议包含触发条件、日志或波形、根因、修复方式和回归验证。
