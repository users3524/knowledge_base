# Seed Resume LaTeX Demo

This is an optional PDF resume demo generated from the Bill Ryan elegant LaTeX template.

Files:

- Source LaTeX: `../../templates/resume-latex/bill-ryan-elegant-zh_CN/seed-topseed-resume.tex`
- Generated PDF: `seed-topseed-resume.pdf`

Important:

- The candidate is synthetic and for demonstration only.
- The wording is intentionally conservative. It avoids fake production, fake metrics, fake ownership, and fake large-scale training.
- Private resumes may inform only high-level layout ideas such as section order and information density. Do not copy personal details, real affiliations, paper titles, project names, metrics, or collaboration traces into this demo.
- Use this as a visual example of how the Doubao Seed final pack can become a PDF-ready resume.

Compile manually:

```bash
cd templates/resume-latex/bill-ryan-elegant-zh_CN
xelatex seed-topseed-resume.tex
cp seed-topseed-resume.pdf ../../../examples/seed-resume-latex/
```
