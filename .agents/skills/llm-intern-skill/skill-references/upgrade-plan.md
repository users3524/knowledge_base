# Upgrade Plan

Goal: give realistic evidence-building tasks, not vague advice.

## Time Buckets

### Half Day

- Clean project README.
- Write ownership boundary.
- Collect screenshots/logs.
- List 10 bad cases.
- Draw pipeline diagram.

### 1 Day

- Build small eval set.
- Add structured logging.
- Add script to reproduce demo.
- Compare one baseline.
- Write a short technical report.

### 3 Days

- Add metrics.
- Add rerank/eval/guardrail/retry module.
- Add tests.
- Run ablation or comparison.
- Produce before/after report.

### 1 Week

- Build complete reproducible repo.
- Add dashboard or demo.
- Add documentation and examples.
- Add multiple baselines.
- Prepare final interview pack.

## Output

Each task should say:

```text
task:
why it matters:
time:
evidence produced:
resume claim unlocked:
interview question it helps answer:
```
