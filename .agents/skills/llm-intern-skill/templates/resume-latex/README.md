# Resume LaTeX Templates

This directory contains LaTeX resume templates used by LLMInternSkill when generating final resume drafts.

## Default Template

Use `bill-ryan-elegant-zh_CN/` as the default Chinese LaTeX resume template.

Source:

- Overleaf template: <https://www.overleaf.com/latex/templates/bill-ryans-elegant-latex-resume/xcqmhktmzmsw>
- Upstream repository: <https://github.com/billryan/resume>

Notes:

- Compile with XeLaTeX.
- The upstream `zh_CN` branch includes large CJK font bundles. They are downloaded locally but ignored by this repository.
- If fonts are missing, use system CJK fonts or download the upstream font bundle locally.
- Do not remove upstream attribution or license files.
