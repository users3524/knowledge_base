Resume Polish

Strengthens LLM, RAG, Agent, post-training, pretraining, AIGC, multimodal, and search ranking project wording without fabricating experience.

Interview Prep

Generates interviewer-style follow-up questions, risk notes, answer cards, and evidence upgrade plans for weak or thin projects.
