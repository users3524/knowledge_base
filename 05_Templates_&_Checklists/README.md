# 模板与工程检查表

这里存放可直接复用的工程模板、检查表、测试用例和评审清单。目标是把经验变成稳定动作，减少重复踩坑。

## 适合收纳

- 单板上电检查表、原理图评审清单、PCB 检查清单。
- 软件功能测试用例、发布检查表、代码评审清单、Bug 复盘模板。
- 项目立项、需求评审、调试记录、实验报告和交付物模板。

## 沉淀建议

- 模板类文件建议用 `模板_主题.md` 命名。
- 检查表类文件建议用 `Checklist_场景.md` 命名。
- 从项目复盘中提炼出来的高频问题，应优先转成检查项。
