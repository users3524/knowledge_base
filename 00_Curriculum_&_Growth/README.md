# 课程大纲与成长规划

这里存放系统化学习路线、课程大纲、技能树和阶段性成长计划。它是知识库的地图层，用来帮助定位后续要补齐的技术主题。

## 使用方式

- 把课程大纲当成技能树目录，具体经验文档则放到硬件工程或软件架构目录。
- 学完或实践过的主题，可以在大纲中标注状态，并链接到对应笔记。
- 新增成长计划时，建议按阶段、目标、产出物和复盘时间组织。
