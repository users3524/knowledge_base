# Knowledge Base Agent Guidance

This repository is a personal engineering knowledge base.

Before adding, moving, renaming, or substantially rewriting knowledge-base content, read
`.agents/constraints.md` and follow it.

Use the `knowledge-base-maintainer` skill for knowledge classification, Markdown organization,
index maintenance, deduplication, or cross-document consistency work.

Preserve unrelated user changes. Do not modify binary engineering project files unless the user
explicitly requests it.

When creating or proposing Git commit messages, follow
`01_Frameworks_&_Flows/Git提交日志规范.md`. Keep Conventional Commit type keywords in English and
write the scope, subject, and body in Simplified Chinese.
