# PCB 布局与结构约束

本文用于原理图完成后、布线开始前的 PCB 结构输入和器件布局。布局的目标不是追求形式上的整齐，而是在机械边界、安全隔离、关键电流环路、信号完整性、热设计和可制造性之间取得可验证的平衡。

## 1. 布局前必须冻结的输入

| 输入 | 需要确认的内容 |
| --- | --- |
| 结构基准 | 坐标原点、单位、板厚、板外形版本和正反面方向 |
| 固定要素 | 安装孔、定位孔、连接器、按键、指示灯、天线和结构接地点 |
| 空间约束 | 禁布区、限高区、螺钉/垫片包络、散热器和线束弯曲空间 |
| 电气边界 | 输入/输出接口、高压网络、隔离边界、机壳地和屏蔽层定义 |
| 热边界 | 主要热源、允许结温、风向、导热路径和温度敏感器件 |
| 工艺边界 | 板厂/贴片厂能力、拼板方式、工艺边和测试夹具要求 |

结构文件、原理图和 PCB 应使用同一版本标识。结构变更后重新执行干涉、安全间距和板边器件检查，不能只凭局部截图确认。

## 2. DXF、STEP 与结构防呆

### 2.1 二维结构数据

- DXF 中分别管理板外形、开槽、安装孔、禁布区、限高区和结构定位线，导入后检查单位、比例、原点和轮廓闭合性。
- 外形和安装孔应来自受控结构源，避免在 PCB 中手工重画后形成两套尺寸基准。
- 连接器、Type-C、网口、接线端子和拨码开关应记录板边突出量、装配公差、插拔空间及线束受力方向。
- 对定位孔、连接器和外壳配合尺寸执行公差叠加，不把标称尺寸直接当成最坏边界。

### 2.2 三维干涉检查

- 连接器、开关、磁性器件、大电容、散热器、屏蔽罩和其他结构关键器件应使用尺寸可信的 STEP 模型。
- 普通小型阻容不必为了形式完整逐一建立高精度模型，但其封装高度和禁布包络应准确。
- 检查器件与外壳、螺钉、支柱、导热垫、线束、相邻板卡及返修工具之间的干涉。
- 3D 检查只能验证几何关系，不能替代公差分析、振动评估和装配实测。

## 3. 功能分区与能量流向

### 3.1 先按电流回路划分区域

优先识别并标注以下区域：

- 电源入口、保护和滤波区域。
- 开关电源、逆变桥和电机驱动等高 $di/dt$ 功率区域。
- MCU、FPGA、存储器和高速数字接口区域。
- ADC、基准源、传感器和小信号模拟区域。
- 射频、天线和时钟区域。
- 高压、隔离和机壳接地区域。

分区的核心是控制电流回路、耦合路径和热影响，不是简单地把数字、模拟区域拉开固定距离。输入、保护、变换和输出通常应沿能量流向布局，但在满足关键回路和机械约束的前提下允许交叉；任何交叉都要明确参考平面和回流路径。

### 3.2 高压与隔离边界

- 电气间隙和爬电距离必须根据适用产品标准计算，不能统一采用 5 mm 或 10 mm。
- 设计输入至少包括工作电压、瞬态过电压、过压类别、污染等级、材料组/CTI、海拔、绝缘类型和涂覆工艺。
- 标准可能来自 IEC 60664-1、IEC 62368-1、IEC 61010-1、IEC 61800-5-1 或其他产品专用规范，应在项目规则中记录标准名称和版本。
- 开槽可以增加表面爬电路径，但开槽尺寸、机械强度和污染风险仍需满足适用标准及板厂能力。
- 隔离边界应同时约束所有层的铜、过孔、测试点、丝印、紧固件和导电结构件。

## 4. 关键器件布局

### 4.1 主控、存储器和高速接口

- MCU/DSP/FPGA 的位置由连接器、存储器、时钟、电源、BGA 扇出和关键接口长度共同决定，不要求机械居中。
- 先放置时钟、外部存储器、高速收发器和电源去耦，再评估剩余扇出通道。
- 每颗去耦电容应形成短而低电感的“电源引脚 -> 电容 -> 地/电源平面”回路；去耦不是串联的“先过电容再进引脚”。
- BGA 器件应在布局阶段确定扇出、过孔工艺和叠层，否则布线阶段可能无法满足阻抗或工艺要求。

### 4.2 接口保护器件

- TVS、ESD 阵列和共模器件应靠近连接器与区域边界，使未保护走线尽可能短。
- 先确定瞬态电流要返回机壳地、屏蔽层、保护地还是信号参考地，再设计短、宽、低电感的泄放路径。
- 保护器件之后的受保护走线不要与连接器侧未保护走线平行或交叉，避免瞬态重新耦合。
- 过孔并非绝对禁止。跨层泄放时可使用多个并联过孔降低电感，但应避免细长单过孔和绕远回路。

### 4.3 晶振与低噪声模拟器件

- 晶振和负载电容靠近对应引脚，走线短且对称，远离 SW 节点、磁性器件、高速时钟和周期性大电流回路。
- 晶振下方避免穿越无关高速信号；是否铺地、包地以及禁止铜的范围以 MCU/晶振厂商参考设计为准，避免额外寄生电容导致起振裕量下降。
- ADC、基准源和高阻节点远离热源及高 $dv/dt$、高 $di/dt$ 区域，并保持连续、可预测的参考平面。
- 分流电阻本身可能是热源。其 Kelvin 采样、热对称和邻近器件温漂应一起评估。

### 4.4 热布局

- 根据损耗估算和热阻路径确定功率器件、铜皮、热过孔、散热器和风道位置。
- 发热器件与晶振、基准源、ADC、精密电阻和温度传感器之间保留足够热隔离，并考虑板内导热而不仅是几何距离。
- 需要壳体导热的器件应同时检查器件高度、公差、导热垫压缩量和板弯风险。

## 5. 布局评审输出

布局冻结前至少输出：

- 结构版本、板框、安装孔、限高和禁布区核对结果。
- 高压/隔离边界及其标准、参数和最小间距计算依据。
- 电源、接口、时钟、模拟和高速区域的关键回路示意图。
- ESD/浪涌泄放路径和机壳地/信号地关系。
- 热源、温敏器件、散热路径和最高风险工况。
- 关键器件 3D 干涉检查结果及遗留风险。
