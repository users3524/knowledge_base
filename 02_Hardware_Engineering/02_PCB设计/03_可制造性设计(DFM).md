# PCB 可制造性与可测试性设计（DFM/DFT）

本文覆盖裸板制造、SMT/波峰焊、拼板和生产测试的设计输入。DFM 的目标不是追求板厂公布的极限能力，而是在成本、良率、返修、可靠性和供应商兼容性之间选择可稳定量产的规则。

## 1. 先建立供应商能力表

投板前向板厂、钢网厂、贴片厂和测试夹具供应商确认能力，并区分“工艺极限”和“量产推荐值”。

| 类别 | 需要确认的参数 |
| --- | --- |
| 图形 | 内/外层推荐及最小线宽线距、铜到板边/槽边距离、蚀刻补偿 |
| 钻孔 | 最小成品孔径、钻径补偿、环宽、孔壁铜厚、纵横比、背钻能力 |
| 阻焊 | 最小阻焊桥、开窗补偿、塞孔/盖油能力、颜色和表面处理影响 |
| 特殊孔 | 盲埋孔、激光孔、树脂填孔、电镀填平和 Via-in-Pad 能力 |
| 叠层 | 材料、铜厚、介质厚度、阻抗公差、翘曲和板厚公差 |
| 装配 | 最小器件间距、钢网厚度、贴装精度、回流/波峰焊能力、返修空间 |
| 测试 | 飞针、ICT、功能测试的探针尺寸、网格、压力、定位和覆盖率能力 |

例如 3.5/3.5 mil 线宽线距、0.2 mm 成品孔或 0.4 mm 焊盘只能作为某家供应商、某种叠层和铜厚下的示例，不能直接写成所有项目的统一标准。

## 2. 裸板制造规则

### 2.1 线宽、间距与孔

- 项目规则优先采用量产推荐值，只有密度确实需要时才局部使用极限能力，并标注位置和风险。
- 过孔设计同时校核成品孔径、焊盘、环宽、板厚纵横比、孔壁铜厚、阻焊方式和可靠性等级。
- 铜到板边、V-Cut、铣槽和金属化槽的距离以板厂公差为依据，不能只保留理论零偏差。
- 阻抗线与普通线的线宽线距由叠层求解结果决定，不能为了满足统一最小线宽随意修改。

### 2.2 过孔表面处理

| 工艺 | 适用场景 | 主要风险/限制 |
| --- | --- | --- |
| 开窗 | 需要测试、散热或焊接的过孔 | 可能吃锡、桥连或暴露铜面 |
| 盖油（Tented Via） | 普通非测试过孔 | 阻焊破裂、单面覆盖和孔内残留需与板厂确认 |
| 塞油 | 防止吃锡或改善阻焊平整度 | 塞孔饱满度、气泡和固化能力受工艺限制 |
| 树脂填孔+电镀填平 | Via-in-Pad、密集 BGA 或需要平整焊盘的区域 | 成本高，需要匹配孔径、纵横比和填孔能力 |

BGA 区域并非一律需要塞孔：

- 焊盘外的普通狗骨扇出孔可按装配要求选择盖油、塞油或开窗。
- 焊盘内过孔（VIPPO）通常需要树脂填孔并电镀填平，避免吃锡和焊点空洞。
- 具体工艺还要结合 BGA Pitch、焊盘尺寸、X-Ray 验收和返修要求。

## 3. SMT 与焊接可靠性

### 3.1 焊盘和钢网

- 优先采用器件厂商推荐封装，并结合适用版本的 IPC-7351、贴装精度和实际焊接工艺复核。
- 片式器件两端焊盘、阻焊、走线和连接铜面积尽量保持热对称，降低立碑和偏移风险。
- QFN、功率器件等裸露散热焊盘通常需要实连铜、合理的热过孔和分窗钢网，焊膏覆盖率及开孔方式按厂商建议和装配验证确定。
- 大铜皮上的通孔或普通焊盘是否使用 Thermal Relief，取决于载流、导热、手焊/波峰焊和回流工艺；不能统一要求所有大引脚和散热焊盘使用梅花连接。
- 高电流、低阻抗节点若使用 Thermal Relief，应校核其截面积、压降、温升和故障电流。

### 3.2 器件排布

- 为贴片吸嘴、AOI、X-Ray、选择焊和返修工具保留可达空间，连接器、屏蔽罩和高器件不能遮挡待检焊点。
- 波峰焊/选择焊器件方向和间距应避免阴影效应、连锡和偷锡不足，具体方向由装配厂设备和载具决定。
- 大质量器件、异形器件和板边连接器应评估贴装固定、振动、跌落及分板应力。
- 温度敏感或易吸湿器件应在生产资料中注明 MSL、烘烤、回流次数和存储要求。

## 4. 拼板、工艺边与分板

- V-Cut、铣槽和邮票孔的尺寸、残厚、孔径、孔距和铜/器件避让由板厂及装配厂共同确认。
- V-Cut 到走线 0.4 mm 只能作为供应商示例；还应考虑 V 槽深度公差、铜箔厚度、器件距板边和分板应力。
- 工艺边应满足传送、夹持、定位、波峰焊方向和测试夹具要求，并避开外壳配合面和关键天线区域。
- 拼板方向和阴阳板方案要评估铜分布、热容量、回流温差、翘曲和器件受力，不能只追求材料利用率。

### 4.1 Fiducial/Mark 点

- 全局和局部 Fiducial 的数量、尺寸、阻焊开窗、表面处理和净空按贴片厂视觉系统要求确定。
- 常见方案是在拼板设置 3 个不共线且不完全对称的全局 Fiducial，细间距 BGA/QFN 附近设置 2 个局部 Fiducial；这只是常见做法，最终以装配厂能力为准。
- Fiducial 周围避免铜皮、丝印、过孔和器件造成视觉干扰，净空尺寸不能统一固定为 1.5 mm。

## 5. 可测试性设计（DFT）

### 5.1 先确定测试策略

根据产品数量、故障覆盖目标和成本选择飞针、ICT、边界扫描、功能测试或组合方案，并明确：

- 必测电源轨、地、复位、时钟、通信、编程和关键模拟节点。
- 开短路、器件值、功能、校准和安全测试分别由哪个工序覆盖。
- 测试点覆盖率、不可测网络、误判率和节拍目标。
- 测试夹具版本、板卡版本和测试程序版本的追溯关系。

### 5.2 测试点设计

- 测试点直径、间距、网格、阻焊开窗、边缘距离和探针方向以夹具供应商能力为准；0.8 mm 只能作为常见起点。
- 优先将测试点放在同一面，避免器件、屏蔽罩、散热器和高压区域阻挡探针。
- 高频、差分和高阻节点增加测试点前应评估 Stub、寄生电容、阻抗不连续和泄漏。
- 为每个电源域提供足够的地针和电源针，避免夹具回路阻抗影响压降、纹波和功能测试结果。
- 编程、边界扫描和校准接口应考虑防呆、掉电恢复、量产权限及序列号追溯。

## 6. 投产前交付

DFM/DFT 评审至少确认：

- 板厂和装配厂能力表已经冻结，极限工艺位置有明确标记。
- Gerber/ODB++、钻孔、叠层、阻抗、表面处理、阻焊和特殊工艺说明一致。
- 封装、钢网、热焊盘、过孔处理和器件方向已完成装配审查。
- 拼板、工艺边、Fiducial、分板方式和板边器件满足生产要求。
- 测试策略、测试点覆盖、夹具接口和版本追溯已经与生产测试方确认。
- 首板、试产和量产阶段的 DFM 问题能够回写到封装库和设计规则，而不是只保留在一次性邮件中。
