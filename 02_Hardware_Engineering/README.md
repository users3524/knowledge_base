# 硬件工程

这里沉淀电路设计、器件选型、调试维修、可靠性验证和硬件问题定位经验。适合放能够帮助硬件设计、评审、调试和量产排错的资料。

## 现有文档

- [嵌入式硬件课程大纲](<嵌入式硬件课程大纲.md>)

## 适合收纳

- 电路设计与仿真。
- 电源、电机驱动与功率器件。
- EMC/ESD、环境可靠性和硬件测试整改。

## 沉淀建议

- 单个芯片、单个电路模块或一次调试记录，优先放入对应子目录。
- 经验文档建议包含问题现象、测试条件、定位过程、根因、解决方案和可复用检查项。
- 能转化为标准动作的内容，可同步提炼到 `05_Templates_&_Checklists`。
